== Description ==

You can display Shopper approved reviews in your website using this Shopper Approved plugin.

= Special Features =

* Pagination of reviews
* change the order of reviews

== Installation ==

1. Upload the entire `Shopper Approved` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Look at your **Setting > SPA settings** and add credentials there then enjoy using **[spa_shortcode]** shortcode...

= Minimum Requirements =

* WordPress 2.1 or higher

= Manual Installation =

1. Unzip the files and upload the folder into your plugins folder (/wp-content/plugins/) overwriting older versions if they exist
2. Activate the plugin in your WordPress admin area.
